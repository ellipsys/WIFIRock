# WIFIRock

Container centered in all the common tools for pentesting wifi networks.

This Docker container has the next tools:

- Macchanger
- Wireshark
- Pyrit
- Aircrack suit
- Wps-pixie
- Hcxdump
- Hcxtools
- Bully
- Hashcat
- Reaver
- Cowpatty
- Wifite
- Fluxion
- All the WPS-Scripts
- Beef

Run command

```
sudo docker run -i -t --network="host" --privileged andresmontoyain/wifirock bash
```
